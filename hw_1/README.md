# Homework 1

This is the NGrams assignment.

## How to run

This assignment uses Ant to compile and jar the program. The CADE machines should have Ant installed. Simply run the
following commands to generate the jar file from the code and then run it:

```
ant compile
ant jar
java -jar build/jar/hw1.jar <your-training-file> -test <your-test-file>
```
